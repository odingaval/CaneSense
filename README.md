# Canesense — Sugarcane Stress Cause Advisor

Canesense helps smallholder sugarcane farmers in Kenya diagnose WHY their crop is stressed — water shortage, soil nutrient deficiency, or pest pressure — by combining what the farmer reports with satellite and soil data for their exact field.

## Setup Instructions

1. **Backend**:
   ```bash
   cd backend
   npm install
   npm run dev
   ```

2. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. Open `http://localhost:5173` for the frontend. The backend runs on `http://localhost:4000`.
